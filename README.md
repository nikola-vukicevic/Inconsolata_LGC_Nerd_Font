# Inconsolata_LGC_Nerd_Font

Inconsolata LGC Nerd Font with different metrics, some modifications (tilde, backtick) and some alignments (braces, parenthesis).

Features:

- Nerd glyphs
- Cyrilic support

Patch made with Font Forge.
